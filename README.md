Декоративний підкреслений псевдоелемент для активного навігаційного лінка реалізовано некоректно: селектор .active-link:active::after:focus є недопустимим (псевдоелемент має бути останнім) і націлюється на стани взаємодії, а не на постійний стан активного лінка, тому підкреслення ніколи не з’являється.

Клас активного стану застосовується занадто широко (.active-link використовується на логотипі, усіх навігаційних лінках і обох контактних посиланнях); його слід застосовувати лише до навігаційного лінка поточної сторінки, щоб лише це посилання мало активне оформлення і підкреслення.

Оформлення активного лінка наразі містить box-shadow на .active-link, хоча візуальний акцент для активного навігаційного лінка має створюватись за допомогою нижньої смуги ::after під лінком, а не тіні навколо елемента.
